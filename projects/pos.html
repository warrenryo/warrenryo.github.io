Point of Sale Management System

I created this project using Laravel with built in authentication using Laravel Auth UI. This Point of Sale Management System allows us to add a specific product, brands and the quantity and it also have a receipt generated in order to track the order easily.

Project Source Code: https://github.com/warrenryo/Point-of-Sale-Management-System

This Project Contains the following technologies used:

Laravel Auth

Bootstrap

Javascript
